Admin Setup
Connect to your free Supabase database to see all student interview records.

One-time setup (5 minutes):
1. Go to supabase.com → Sign up free
2. Create a new project (any name, remember the password)
3. Go to SQL Editor → paste and run this SQL:
CREATE TABLE IF NOT EXISTS health_survey_records (
  id bigserial primary key,
  record_id text unique,
  interviewer text,
  interview_date date,
  location text,
  respondent_age int,
  respondent_gender text,
  house_type text,
  water_source text,
  water_treated text,
  latrine text,
  hiv_heard text,
  hiv_tested text,
  illnesses text,
  deaths_5yr text,
  deaths_count int,
  fuel text,
  education text,
  occupation text,
  consent text,
  synced_at timestamptz,
  raw_json jsonb
);
ALTER TABLE health_survey_records ENABLE ROW LEVEL SECURITY;
CREATE POLICY "anon insert" ON health_survey_records FOR INSERT TO anon WITH CHECK (true);
CREATE POLICY "admin select" ON health_survey_records FOR SELECT TO anon USING (true); 4. Go to Settings → API → copy the Project URL and anon public key
5. Paste them below, then share the URL + key with your students too (for their sync)

Your URL and key are stored only on this device. No one else can access this dashboard without your password.
