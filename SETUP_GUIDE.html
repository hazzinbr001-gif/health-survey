📲 Install Health Survey as a Phone App

Step-by-step guide — Free GitHub hosting in under 10 minutes

📦 Files in this package
index.html ← The full survey app
manifest.json ← PWA config
sw.js ← Offline service worker
icon-192.png ← App icon
icon-512.png ← App icon (large)
SETUP_GUIDE.html ← This file

⚠ Keep ALL files in the same folder. Do not rename them.

1️⃣ Create a Free GitHub Account
1
Go to github.com on your phone or computer Tap "Sign up" — it's completely free
2
Choose a username Example: hazzinbr — this becomes part of your app's web address
3
Verify your email GitHub will send a confirmation email — click the link
2️⃣ Create a New Repository
1
Click the "+" button → "New repository" Top right corner of GitHub
2
Name it exactly: health-survey This name becomes your URL
3
Set to Public Must be Public for free GitHub Pages hosting
4
Leave everything else default → Click "Create repository" Do NOT tick "Add README"
3️⃣ Upload All 5 Files
1
On the empty repo page, click "uploading an existing file" Or drag and drop the files directly
2
Upload ALL 5 files at once index.html · manifest.json · sw.js · icon-192.png · icon-512.png
3
Scroll down → Click "Commit changes" The green button at the bottom of the page
4️⃣ Turn On GitHub Pages (Free Hosting)
1
Go to your repository → Click "Settings" Settings tab is near the top of the repo
2
Scroll down to "Pages" in the left sidebar Under "Code and automation"
3
Under "Branch" — select "main" → Click Save Leave the folder as / (root)
4
Wait 1–2 minutes, then your app is live at:
https://YOUR-USERNAME.github.io/health-survey/
Replace YOUR-USERNAME with your actual GitHub username
5️⃣ Install App on Android Phone
1
Open Chrome on your Android phone Must be Chrome — not Samsung browser or Firefox
2
Go to your app URL
https://YOUR-USERNAME.github.io/health-survey/
3
A banner will pop up: "Add Health Survey to Home screen" Tap "Install" or "Add" — Chrome shows this automatically on HTTPS
4
If no banner: tap ⋮ (three dots) → "Add to Home screen" Or tap the install icon ⊕ in the address bar
5
✓ Done! The app opens like a native app — no browser bar Works fully offline after first load. Data saved on phone only.
🍎 Install on iPhone
1
Open in Safari (not Chrome) iPhone PWA install only works in Safari
2
Tap the Share icon ⬆ at the bottom
3
Scroll down → tap "Add to Home Screen"
4
Tap "Add" → App icon appears on home screen
📤 Share with Your Team

Once hosted, share your URL with all interviewers via WhatsApp:

https://YOUR-USERNAME.github.io/health-survey/

Everyone on the team can install it on their own phone from the same link. Each phone stores its own data separately (offline).
